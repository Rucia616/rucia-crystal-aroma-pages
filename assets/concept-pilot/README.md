Project: 玄学晶石香薰 / Rucia Crystal Aroma  
Stage: Concept validation  
Status: GREEN, can run remotely from China  
Recommended platform: PickFu

## Why This Test Exists

Before spending money on samples, packaging, creators, or ads, this pilot checks one dangerous question:

Can North American consumers quickly understand and care about a light-mystic fragrance product?

This is not a scent test. This is a concept comprehension and appeal test.

## PickFu Setup

Use this as the first paid pilot only after the cash guardrails below are accepted. This is not a foolproof test. It is a low-cost filter for obvious concept confusion.

- Safest first format: one or two Head-to-Head image polls. Start with A vs C; only run B vs D if the first result is useful and we still need to understand stronger mystic language.
- Optional faster format: 4-option Single Select image poll, but only if the final quote is acceptable.
- Sample: minimum practical pilot the platform allows, ideally 20-30 U.S. respondents per small poll. If results are promising, rerun with 100.
- Audience: U.S., women 18-34 if available at acceptable cost. If too narrow or expensive, use U.S. women 18-44. If interest traits such as home fragrance, candles, astrology, self-care, or wellness are available, add no more than 1-2 traits so the test does not become too narrow.
- Budget guardrail: first paid spend should be capped at $80 unless Rucia explicitly approves more. PickFu starts at $1 per response and $15 per poll; final cost changes with poll type, targeting, and add-ons. If the builder shows more than $80 for the first pilot, do not launch yet; reduce targeting, reduce options, or run one Head-to-Head first.
- Output: winner share, respondent explanations, negative-language themes, clarity coding.

## Exact Workflow

1. Open PickFu and create a new poll.
2. Start with Head-to-Head A vs C if Rucia wants the lowest-risk spend. This compares `scent sign` against `room mood`, the two most commercially plausible routes.
3. If A vs C has a clear winner and comments are useful, run B vs D only if we still want to understand stronger mystic language. Otherwise skip B vs D.
4. Use 4-option Single Select only if the quote is still comfortably under the cash guardrail.
5. Upload the PNG boards from `boards/`.
6. Paste the main question and follow-up prompts from `pickfu-questionnaire.md`.
7. Preview before publishing. Do not launch if the images appear cropped, different sizes, or out of order.
8. Launch only if the final quote is within the cash guardrail.
9. Export results and paste/copy respondent comments into `results/pickfu-results-template.csv`.
10. Score using `scoring-rubric.csv`.
11. Decide: continue, revise, or stop.

## Pass Gate

Continue only if all of these are true:

- Category clarity: 75%+ can describe it as fragrance, diffuser, aroma, scent, or room scent.
- Winner signal: the top concept gets 35%+ of votes and beats the runner-up by at least 10 percentage points in the 4-way pilot.
- Non-cringe: fewer than 20% of comments use negative themes such as scam, fake, too witchy, confusing, childish, cheap, or gimmicky.
- Scent-forward appeal: 50%+ mention scent, smell, fragrance, aroma, or room feeling as the appeal.
- Quiz comprehension: 40%+ understand that quiz/match/result is part of the purchase path.

## Fail Actions

- If people do not understand the category: remove crystal/astrology emphasis and make the product visibly a diffuser.
- If people like the look but not the concept: keep VI, rewrite offer language.
- If mystic language creates skepticism: shift toward `room mood`, `self-care ritual`, and `giftable fragrance`.
- If the winner is unclear: run a 100-person follow-up before spending on physical samples.
- If all options feel gimmicky: pause mystic positioning and test a fragrance-first angle.

## What This Does Not Prove

This test does not prove:

- people like the actual scent
- people will repurchase
- the price is acceptable
- ads will convert profitably
- TikTok creators can explain it naturally
- packaging will survive U.S. shipping
- fragrance/liquid shipping is compliant
- the PickFu panel perfectly represents TikTok/Pinterest shoppers
- the winning board will stay the winner after real product photos, price, and landing page copy are added

Those require later stages: price test, scent sign blind test, U.S. sample node, and creator seeding.

## Blind Spots To Watch

- Panel mismatch: PickFu respondents are useful for fast feedback, but they are not guaranteed to be active TikTok beauty/wellness buyers.
- Concept-board bias: respondents may react to color, layout, or wording rather than the underlying product idea.
- No scent evidence: fragrance preference and refill intent cannot be inferred from images.
- Sample-size noise: 20-30 responses can catch obvious problems, but small differences are not reliable.
- Question framing risk: asking too much about astrology may overstate mystic interest; asking too much about purchase may overstate buying intent.

If these blind spots bother us after previewing the quote, pause the spend and move to a free/near-free review of the test setup before launch.

## Next Step After Results

If one concept passes, build a 1-page landing page around that winning concept and run:

- 100-person PickFu follow-up or Prolific survey for language depth
- 5-second landing comprehension test
- Van Westendorp price test
- U.S. in-home scent test after sample node is ready
