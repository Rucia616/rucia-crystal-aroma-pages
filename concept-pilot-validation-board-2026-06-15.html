How To Run

PickFu Setup

  1. 先创建 Head-to-Head image poll：A vs C。
  2. 只上传 A/C 两张 PNG，不要改图。
  3. 受众先选美国女性 18-34，贵了就改 18-44。
  4. 最终报价超过 80 美元就不发起。
  5. A/C 有清晰结果后，再决定是否花钱测 B/D。
  6. 导出结果，用模板编码 clarity / scent / quiz / gift / negative flags。
Pass Gate

Continue Only If

75%+能说清这是 fragrance / diffuser / room scent / aroma。
55%+A/C 胜出方至少拿到 55%，也就是领先另一方至少 10 个百分点。
<20%负面词如 scam / fake / too witchy / confusing / childish 不超过 20%。
50%+吸引点来自 scent / smell / fragrance / room feeling，而不只是晶石。
40%+理解 quiz / match / result 是购买路径的一部分。
Not Foolproof

Blind Spots

  1. PickFu 受访者不等于 TikTok / Pinterest 香氛买家。
  2. 概念板测试会混入视觉、文案、颜色偏好，不是纯产品测试。
  3. 图片无法证明香味好闻、留香、扩香和复购。
  4. 50 人也只能抓明显问题，小差距不能当结论。
  5. 购买意愿回答不等于真实下单。
Cash Discipline

Insurance Rules

先 A/C先比 `Scent Sign` 和 `Room Mood`，这是最可能商业化的两条。
不加项不要为了平台推荐的 add-ons 超预算。
不硬判如果差距小、解释弱，就当没结论，不强行选方向。
不跳级概念没过，不做样品、不找红人、不投广告。
